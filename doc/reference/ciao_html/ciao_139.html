PREDICATE: coord/4:

Usage: coord(X1, Y1, X2, Y2)

  • Description: X1, Y1, X2, and Y2 give the coordinates of two diagonally opposite corners of a rectangular region enclosing the oval.
  • Call and exit should be compatible with: X1 is an integer. (basic_props:int/1) Y1 is an integer. (basic_props:int/1) X2 is an integer. (basic_props:int/1) Y2 is an integer. (basic_props:int/1)
  • The following properties should hold at call time: X1 is currently a term which is not a free variable. (term_typing:nonvar/1) Y1 is currently a term which is not a free variable. (term_typing:nonvar/1) X2 is currently a term which is not a free variable. (term_typing:nonvar/1) Y2 is currently a term which is not a free variable. (term_typing:nonvar/1)
