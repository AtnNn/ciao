Documentation on exports (between)

PREDICATE: between/3:

Usage: between(?N, +Min, +Max)

  • Description: N is an integer which is greater than or equal to Min and smaller than or equal to Max. Both Min and Max can be either integer or real numbers.
  • The following properties should hold at call time: ?N is an integer. (basic_props:int/1) +Min is currently instantiated to a number. (term_typing:number/1) +Max is currently instantiated to a number. (term_typing:number/1)
