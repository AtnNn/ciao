Usage and interface (menu_class)

  • Library usage: :- use_module(library(menu_class)).
  • Exports:
    • Predicates: name_menu/1, menu_data/1, label_value/1, tearoff_value/1, tcl_name/1, creation_options/1, creation_options_entry/1, creation_menu_name/1.
  • Other modules used:
    • System library modules: objects/objects_rt, tcltk_obj/window_class, tcltk_obj/menu_entry_class, tcltk/tcltk, tcltk/tcltk_low_level, lists.

Documentation on exports (menu_class)

PREDICATE: name_menu/1:

Usage: name_menu(Name)

  • Description: Indicates the Name of the menubutton associated.
  • Call and exit should be compatible with: Name is currently instantiated to an atom. (term_typing:atom/1)
  • The following properties should hold at call time: Name is currently a term which is not a free variable. (term_typing:nonvar/1)
