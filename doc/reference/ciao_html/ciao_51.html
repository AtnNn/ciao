Enumeration of integers inside a range

Author(s): The CLIP Group..

Version: 1.11#222 (2004/5/24, 13:8:7 CEST)

Version of last change: 1.11#159 (2004/1/9, 16:54:42 CET)

This modules enumerates integers between two numbers, or checks that an integer lies within a range

  • Documentation on exports (between)

Usage and interface (between)

  • Library usage: :- use_module(library(between)).
  • Exports:
    • Predicates: between/3.

Documentation on exports (between)

PREDICATE: between/3:

Usage: between(Min, Max, N)

  • Description: N is an integer which is greater than or equal to Min and smaller than or equal to Max. Both Min and Max can be either integer or real numbers.
  • The following properties should hold at call time: Min is currently a term which is not a free variable. (term_typing:nonvar/1) Max is currently a term which is not a free variable. (term_typing:nonvar/1) Min is currently instantiated to a number. (term_typing:number/1) Max is currently instantiated to a number. (term_typing:number/1) N is an integer. (basic_props:int/1)
