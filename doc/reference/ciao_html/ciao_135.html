Documentation on exports (menubutton_class)

PREDICATE: menu_name/1:

Usage 1: menu_name(Menu)

  • Description: Menu posted when menubutton is clicked.
  • Call and exit should be compatible with: Menu is currently instantiated to an atom. (term_typing:atom/1)
  • The following properties should hold at call time: Menu is currently a term which is not a free variable. (term_typing:nonvar/1)

Usage 2: menu_name(Menu)

  • Description: Gets the name of the Menu asociated to the menubutton.
  • Call and exit should be compatible with: Menu is currently instantiated to an atom. (term_typing:atom/1)
  • The following properties should hold at call time: Menu is a free variable. (term_typing:var/1)
