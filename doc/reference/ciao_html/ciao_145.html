Low-level socket interface to SQL/ODBC databases

Author(s): Jose Morales.

Version: 1.8#1 (2002/5/27, 19:57:48 CEST)

Version of last change: 1.7#209 (2002/4/24, 14:50:46 CEST)

This library provides a low-level interface to MySQL using the MySQL C API and the Ciao foreign interface to C.

  • Usage and interface (mysql_client)
  • Documentation on exports (mysql_client)

Usage and interface (mysql_client)

  • Library usage: :- use_module(library(mysql_client)).
  • Exports:
    • Predicates: mysql_connect/5, mysql_query/3, mysql_query_one_tuple/3, mysql_free_query_connection/1, mysql_fetch/2, mysql_get_tables/2, mysql_table_types/3, mysql_disconnect/1.
    • Regular Types: dbconnection/1, dbqueryconnection/1.
  • Other modules used:
    • System library modules: foreign_interface/foreign_interface_properties, persdb_mysql/db_client_types.
