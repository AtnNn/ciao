This library package allows the use of the ISO-Prolog predicates in Ciao programs. The compatibility is not at 100% yet.

Usage and interface (iso)

    • System library modules: aggregates, dynamic, iso_misc, iso_byte_char, iso_incomplete, operators, read, write.
