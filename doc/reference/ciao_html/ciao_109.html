Connecting with the Web (PiLLoW)

This package implements the PiLLoW library [CHV96a] in Ciao. It is not yet documented here, but you can access http://clip.dia.fi.upm.es/Software/pillow/pillow.html to see documentation on previous versions. However, note that this version is newer and has some bug fixes and added functionality (templates, XML processing, and others, which you can try to guess the rest by looking at the source). You can also find a paper and some additional information in the library/pillow/doc directory of the distribution.

Usage and interface (pillow)

  • Library usage: :- use_package(pillow). or :- module(...,...,[pillow]).
  • New operators defined: $/2 [150,xfx], $/1 [150,fx].
  • Other modules used:
    • System library modules: pillow/http, pillow/html.
