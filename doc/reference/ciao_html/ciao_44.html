Classic Prolog package

Author(s): The CLIP Group.

Version: 1.11#120 (2003/12/22, 23:35:35 CET)

Version of last change: 1.11#85 (2003/12/21, 2:0:26 CET)

This library package allows the use of certain Prolog features which have become sort of 'classical' from many Prolog implementations. These include definite clause grammars and some classical predicates like append/3. The libraries listed below define these predicates, and the following chapters describe them.

Usage and interface (classic)

  • Library usage: :- use_package(classic). or :- module(...,...,[classic]).
  • New operators defined: -->/2 [1200,xfx].
  • Other modules used:
    • System library modules: operators, old_database, lists, sort, dict, strings, dec10_io, format, ttyout, compiler/compiler.
