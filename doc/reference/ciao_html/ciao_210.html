Documentation on exports (dictionary)

PREDICATE: dictionary/6:

Usage 1: dictionary(?NodeTypeId, ?AccessType, ?FieldTypeId, ?FieldId, -Init_value, -Boundary)

  • Description: To lookup information about the nodes, getting their properties. Note that the type returned for the bound can be of two different types bound or bound_double. The rotation type have one bound for the directions and one for the degree of rotation.
  • Call and exit should be compatible with: ?NodeTypeId is an atom. (basic_props:atm/1) ?AccessType is an atom. (basic_props:atm/1) ?FieldTypeId is an atom. (basic_props:atm/1) ?FieldId is an atom. (basic_props:atm/1) -Init_value is a list of atms. (basic_props:list/2) -Boundary is a variable interval. (internal_types:bound/1)

Usage 2: dictionary(?NodeTypeId, ?AccessType, ?FieldTypeId, ?FieldId, -Init_value, -Boundary)

  • Description: To lookup information about the nodes, getting their properties. Note that the type returned for the bound can be of two different types bound or bound_double. The rotation type have one bound for the directions and one for the degree of rotation.

(basic_props:atm/1) -Init_value is a list of atms. (basic_props:list/2) -Boundary is a variable interval. (internal_types:bound_double/1)
