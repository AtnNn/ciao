Documentation on exports (atom2term)

PREDICATE: atom2term/2:

Usage: atom2term(+Atom,-(Term))

  • Description: Convert an atom into a term. Atom is an atom, but must have term syntax. Term is a term resulting from parsing Atom char by char. Note: this is just a simple version written for parsing daVinci's messages. There should be a call to the standard reader to do this!

PREDICATE: string2term/2:

Usage: string2term(+String,-(Term))

  • Description: Same as atom2term/2 but first argument is a string (containing a term).
